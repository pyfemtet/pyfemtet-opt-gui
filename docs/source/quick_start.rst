Quick Start
===========

Running the GUI
---------------
To start the GUI, run:

.. code-block:: bash

   python -m pyfemtet_opt_gui

Basic Workflow
--------------
1. Open a Femtet project file.
2. Set up optimization parameters.
3. Run the optimization.
4. Review results.
